Aide
Comment utiliser :
  1. Sélectionnez le type de rapport souhaité
  2. Choisissez les paramètres si nécessaire (étudiant, cours)
  3. Cliquez sur "Télécharger" pour générer le PDF
Formats disponibles :

PDF Tous les rapports sont générés au format PDF et peuvent être imprimés ou partagés.
